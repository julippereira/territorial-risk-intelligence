# 📊 Power BI Dashboard

This directory contains the Power BI dashboard developed for the **Territorial Risk Intelligence** project.

The dashboard integrates demographic, socioeconomic, housing, environmental, and geospatial indicators to support household-level risk assessment, territorial prioritization, and evidence-based decision-making.

---

# 📁 File

```text
territorial_risk_intelligence.pbix
```

---

# 🎯 Dashboard Overview

The dashboard consolidates the outputs generated by the analytical models available in the `analytics/` directory and transforms them into interactive territorial intelligence visualizations.

Analytical workflow:

```text
Household Survey Data
            ↓
Data Preparation
            ↓
Data Normalization
            ↓
Social Vulnerability Index
            ↓
Environmental Risk Index
            ↓
Socioenvironmental Matrix
            ↓
Power BI Dashboard
```
